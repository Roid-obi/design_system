# Margin

Adds standard margin amount for a given side or axis.

Valid sides/axes are (`all`, `top`, `right`, `bottom`, `left`, `x`, `y`)

Valid amounts are (`xxs`, `xs`, `s`, `m`, `l`, `xl`).
